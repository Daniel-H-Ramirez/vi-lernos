# Vi Lernos - Plataforma Educativa Inteligente

Plataforma educativa con IA que recomienda cursos personalizados usando arquitecturas distribuidas.

## 🚀 Características

- Recomendaciones personalizadas con IA
- Arquitectura de microservicios
- Bases de datos distribuidas (MongoDB + PostgreSQL)
- Comunicación en tiempo real
- Frontend en React
- Backend en FastAPI

## 🛠️ Tecnologías

- **Frontend**: React, Material-UI
- **Backend**: FastAPI, Python
- **Bases de datos**: MongoDB Atlas, PostgreSQL
- **IA**: Scikit-learn, TF-IDF, k-NN
- **Auth**: JWT, OAuth2
